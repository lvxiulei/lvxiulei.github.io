在 Kotlin 中，有五个常用的作用域函数，它们是: apply、also、let、with 和 run。

  1. apply：适用于初始化对象的场景或需要在对象上执行多个操作的场景。 apply 函数返回对象本身。

    val person = Person().apply {
    name = "John"
    age = 25
}

  2. also：适用于需要执行一些附加操作的场景，并且不需要引用对象本身的返回值。 also 函数返回对象本身。

    val list = mutableListOf<Int>()
list.also {
    it.add(1)
    it.add(2)
    it.add(3)
}.also {
    println("List size: ${it.size}")
}

  3. let：适用于对非空对象执行代码块的场景，并且需要引用代码块的结果。 let 函数返回代码块的最后一个表达式的结果。

    val name: String? = "John"
name?.let { nonNullName ->
    println("Name: $nonNullName")
}

  4. with：适用于对非空对象执行多个操作的场景，而无需引用对象本身或对返回值感兴趣。 with 是一个顶层函数，不是作为对象的成员函数。

    val person = Person()
with(person) {
    name = "John"
    age = 25
    println("Person: $name, $age")
}

  5. run：适用于链式操作，并且需要引用对象自身作为返回值。 run 函数返回代码块的最后一个表达式的结果。

    val person = Person()
val updatedPerson = person.run {
    name = "John"
    age = 25
    this
}

    对于with，T.run，T.apply接收者是this，而T.let和T.also接收者是it；对于with,T.run,T.let返回值是作用域的最后一个对象（this）,而T.apply和T.also返回值是调用者本身(itself)。
